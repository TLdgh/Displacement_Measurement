Displacement Measurement

Summary

The goal is to measure the height of a building by measuring the acceleration of an object from the first floor to the last floor.

Suppose the displacement is x(t), then the velocity and the acceleration are: \[\begin{gather*} v(t)=\frac{dx}{dt}\\ a(t)=\frac{dv}{dt} \end{gather*}\]

Therefore the displacement can be calculated as: \[\begin{gather*} x(t)=\int_0^t v(s) ds + C_1\\ v(t)=\int_0^t a(s) ds + C_2 \end{gather*}\]

Given the initial conditions a(0), \(C_2=v(0)=:v_0\), and \(C_1=x(0)=:x_0\), thus we have: \[\begin{gather*} x(t)=\int_0^t v(s) ds + x_0\\ v(t)=\int_0^t a(s) ds + v_0 \end{gather*}\]

Method

Using the accelometer of a cell phone, we can measure the acceleration of an elevator at any time t. This gives us a time series of the acceleration a(t). Thus the displacement is simply the numerical double intergral of the time series. The initial conditions are \(v_0 = x_0 = 0\).

Now we can build the function that calculates the displacement:

def AVDMeasure(Data):
    NewData = Data.iloc[:, [0, 3]]  # create the data with relevant columns
    NewData.columns = ["Time", "Acceleration"]

    # Centralize the acceleration data:
    NewData = NewData.copy()
    NewData.loc[:, "Acceleration"] = (
        NewData["Acceleration"] - NewData["Acceleration"].mean()
    )

    # Add initial conditions
    zeros_row = pd.Series([0, 0], index=NewData.columns)
    NewData = pd.concat([pd.DataFrame(zeros_row).T, NewData]).reset_index(drop=True)

    NewData["Velocity"] = 0
    NewData["Displacement"] = 0

    # Calculate numerical integrals:
    for i in range(1, len(NewData)):
        NewData.at[i, "Velocity"] = NewData.at[i - 1, "Velocity"] + (
            NewData.at[i - 1, "Acceleration"] + NewData.at[i, "Acceleration"]
        ) / 2 * (NewData.at[i, "Time"] - NewData.at[i - 1, "Time"])

        NewData.at[i, "Displacement"] = NewData.at[i - 1, "Displacement"] + (
            NewData.at[i - 1, "Velocity"] + NewData.at[i, "Velocity"]
        ) / 2 * (NewData.at[i, "Time"] - NewData.at[i - 1, "Time"])

    return NewData
